# Soul++ 油猴脚本

## 介绍
让某论坛变得更加顺手和好用一些

## 功能
1. 购买免刷新
2. 无缝加载板块帖子列表
3. 无缝加载贴内楼层列表
4. SFW安全模式（可通过鼠标滑入查看原内容）
    - 折叠贴内图片
    - 替换所有用户头像为默认头像
5. 自动领取和完成日常任务
